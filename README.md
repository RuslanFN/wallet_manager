# Менеджер кошельков

Сервис управления кошельками с возможностью заполнения базы данных тестовыми данными и тестирования.

## Начало работы

Для запуска приложения со всеми зависимостями выполните команду:

```bash
docker-compose up --build
```

Эта команда выполнит следующие действия:
1. Соберет образ Docker для wallet-manager
2. Запустит контейнер PostgreSQL
3. Выполнит миграции базы данных (Alembic)
4. Заполнит базу данных начальными данными
5. Запустит тесты
6. Запустит сервер приложения

## Тестовые данные

В базе данных автоматически создаются два тестовых кошелька:
- ID кошелька: 11111111-1111-1111-1111-111111111111
- ID кошелька: 22222222-2222-2222-2222-222222222222

Эти тестовые кошельки создаются автоматически при запуске приложения и доступны для целей тестирования.

## Структура проекта

Проект использует FastAPI для веб-сервиса, SQLAlchemy для ORM и Alembic для миграций базы данных.
